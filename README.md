# Tkinter UI
A simple light-weight standalone Tkinter framework.

**NOTE:** It just wraps the `tkinter` module and not `tkinter.ttk`.

## Features
### Window
- Like the main container.

### Widgets
- Label
- Button
- Entry
- ComboBox
- SpinBox
- CheckBox
- RadioButtons
- Scrollable
- Slider

### Layouts
- Absolute Layout
- Grid Layout
- Flex Layout

### Styles and Theming
- All provided by Tk except the ones mention in `styles.py`.
