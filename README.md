# Tkinter UI
A simple light-weight standalone Tkinter framework.

**NOTE:** It has nothing to do with `tkinter.ttk`, just the `tkinter` module.

## Features
### Window
- Like the main container.

### Widgets
- Label
- Button
- Entry
- ComboBox
- SpinBox
- CheckBox

### Layouts
- Absolute Layout
- Grid Layout
- Flex Layout

### Styles and Theming
- All provided by Tk except the ones mention in `styles.py`.
